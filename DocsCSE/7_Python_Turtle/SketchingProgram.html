Procedure

1.       Review basic turtle commands for use in your drawing program.

 

Basic Turtle Commands

def()

creates a function call

fd()

moves turtle forward

bk()

moves the turtle backwards

lt()

turns the turtle left

rt()

turns the turtle right

color()

changes color of line

pu()

stops drawing

pd()

continues drawing

goto()

relocates the turtle to screen location

circle()

draws a circle

clear()

clears screen

reset()

resets screen
